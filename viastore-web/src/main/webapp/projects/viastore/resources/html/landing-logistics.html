Логистика

viad@t — многопрофильная система управления внутренней логистикой

Мы разработали мощную систему, которая обеспечит эффективную работу как на небольших складах с малым уровнем товарооборота, так и в дистрибуционных центрах с тысячами наименований товаров в заказе в сутки.

Программное обеспечение для всех направлений внутренней логистики

  • viad@t ONEviad@tONE – предварительно сконфигурированная система управления логистикой склада, включая систему управления движением материалов для малых складов
  • viad@t CLMviad@tCLM – интеллектуальная система по распределению и управлению местами хранения для единиц загрузки на автоматическом складе, работающая под управлением ERP или системы другого производителя
  • viad@t CCSviad@tCCS – эффективное управление подъемно-транспортным оборудованием
  • viad@t MFCviad@tMFC – интеллектуальная система управления движением материалов
  • viad@t VISUviad@tVISU – визуализация работы оборудования для большей наглядности и ускорения доступа
  • viad@t RFMviad@tRFM – эффективная система управления для складов и дистрибуционных центров с любым уровнем товароооборота, обслуживаемых вручную или погрузчиком
  • viad@t SRMviad@tSRM – современное управление стеллажными штабелёрами с высоким уровнем сервиса
  • viad@t SORTviad@tSORT – система управления для сортировочного устройства высокой производительности

Возможности, доступные всем

Полностью масштабируемым является пакет программного обеспечения. Для решения каждой задачи рассчитывается наилучшая стратегия, чтобы максимально оптимизировать процессы складской логистики.

viad@t WMS работает на всех платформах UNIX и Windows. Программа не зависит от операционных систем; Вы можете использовать любую ведущую систему для хранения данных (DB2, MS SQL Server или Oracle).

Ваш персонал всегда сможет оперативно получить достоверную информацию в режиме «онлайн».

viad@t подойдёт для высокоавтоматизированных систем и для традиционного склада, обслуживаемого вручную. Независимо от типа склада система выбирает оптимальное место складирования для товаров с любым оборотом, обеспечивая управление складскими запасами и оптимизируя внутреннюю логистику.

Мы подбираем решения индивидуально, учитывая специфические требования каждого Заказчика. С viad@t управление внутренней логистикой возможно во всех странах мира, благодаря широким языковым настройкам. Программное обеспечение будет расширено вместе с Вашими процессами, Вашим предприятием с согласно Вашим требованиям.

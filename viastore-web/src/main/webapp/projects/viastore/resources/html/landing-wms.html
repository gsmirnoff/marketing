Возможности, доступные всем

WMS viad@t является многоязычным ПО и не зависит от СУБД, аппаратного обеспечения и операционных систем. Работайте с любыми базами данных (Oracle, MS SQL Server, DB2/6000), система имеет версии клиентского ПО для всех популярных ОС (Windows, Linux, AIX...)

Программа WMS содержит все необходимые модули для управления стеллажными штабелерами, подъемно-транспортным оборудованием и другими механическими компонентами.

Она подойдёт для высокоавтоматизированных систем и для традиционного склада, обслуживаемого вручную. Независимо от типа склада система выбирает оптимальное место складирования для товаров с любым оборотом, обеспечивая управление складскими запасами, оптимизируя загрузку и перемещения на складе. Осуществляется поддержка всех видов комиссионирования.

Система дополнительно присоединяется к продуктам других компаний — таких, как специальное экспедиторское, таможенное или отгрузочное программное обеспечение.
